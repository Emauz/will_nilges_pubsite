How to set up networking on an uncooperative CLI.

I need to get to my other thonkpad and get some code...

Until then, you need to add the device to `/etc/network`.

Add the device to your `interfaces` file which might look something like this:

# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

source /etc/network/interfaces.d/*

# The loopback network interface
auto lo
iface lo inet loopback

You'll need to find the device name using ifconfig -a and thenadd the device to this like this: iface *DEVICE NAME* inet dhcp

Then, you'll need to hit it with an ifdown *DEVICE NAME* and then an ifup *DEVICE NAME*
