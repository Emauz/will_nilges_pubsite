Bettervent

Bettervent is a scheduling app that I am developing for Computer Science House. We have tablets outside our special rooms that run this software. It displays the current time, which event is currently taking place, and which events are coming up. It also lets people make short, quick reservations for if they need to snag it for a bit.

Bettervent is open source, and you can find my repo on GitHub. You can also look at Reservator that I tried to fork from, but that app ended up being an overengineered undocumented nightmare, so I tossed everything out and started writing my own from scratch because... well... Just look through the Reservator code and you'll understand :) Currently, I'm working on the fragment-trial branch, which I made to test out a different way of switching between the three features of the app. Check out the README for more information.

BOA

BOA is a Unity game that I am currently developing in C#. It is a fast-paced, PVP, multiplayer shooter with a heavy focus on mobility and reaction. I drew inspiration from Terraria and OpenArena. Terraria PVP is a lot of fun in the 2D voxel-like environments that comprise that game, but I'd like it to be faster, and more focused on the combat. OpenArena is a lot like that, and also captures the general feel I want for the game.

I've had to put it on hold for a while as I focus on school, work, and my projects for CSH, but it'll (hopefully) be done by next year.

Hostile Takeover

Hostile Takeover is a real time strategy Android game I am developing. You, as a company, must buy out all of your competitors before they get big enough to buy you out. It's got most of its game mechanics done. All I need to do is dev the UI and it should be out and ready for play!
